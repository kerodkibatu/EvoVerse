# Reference material

The `TEST.GEL` planaria genome is a recreation of a cellular-development model I
followed from a two-part video walkthrough on building a multicellular organism
out of morphogen gradients, plus a paper describing the gradient diffusion/decay
model.

The original transcripts and paper are **not redistributed here** to respect
their authors' copyright — please find and read/watch the originals from their
source.

`MorphogenGradViz.png` is kept as a quick visual reference for how a single
morphogen's concentration falls off with distance from its source.
